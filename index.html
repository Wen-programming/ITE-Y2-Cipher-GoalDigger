I. Classical Cryptography

I.I. Caesar Cipher

Classical cryptography refers to early encryption techniques developed before the modern digital era. These methods were widely used in military, political, and diplomatic communication to protect secret messages. Unlike modern cryptography, classical systems rely on simple mathematical transformations such as shifting or substituting letters.

The Caesar Cipher is one of the earliest and simplest encryption techniques. It was famously used by Julius Caesar to send secret military instructions to his generals. This cipher works by shifting each letter in the plaintext by a fixed number of positions in the alphabet.

For example, when the shift key is 3: A becomes D, B becomes E, C becomes F. When the alphabet reaches Z, it wraps around back to A. Source

Image: Caesar Cipher Wheel

Emperor Julius Caesar made his messages secret by shifting each letter three positions forward in the alphabet.

Example ->
Plaintext: HELLO
Shift Key: 3
Ciphertext: KHOOR

Mathematical Formula:

Encryption: C = (P + k) mod 26
Decryption: P = (C - k) mod 26

Where:
P = Position of plaintext letter (A=0, B=1, ... Z=25)
C = Position of ciphertext letter
k = Shift key
26 = Number of letters in the English alphabet

I.II. General Shift Cipher

The General Shift Cipher is an extension of the Caesar Cipher. Instead of always shifting letters by 3 positions, the shift value can be any integer between 1 and 25.

In fact, the Caesar Cipher is a special case of the General Shift Cipher where the key k = 3.

In this cipher, each letter of the plaintext is shifted forward by a chosen key value k. The encryption and decryption processes follow the same mathematical formula:

Encryption: C = (P + k) mod 26
Decryption: P = (C - k) mod 26

Although this method offers more flexibility than the Caesar Cipher, it remains insecure because it can be broken using brute force or frequency analysis techniques.

II. RSA Cryptography

RSA (Rivest-Shamir-Adleman) is a public-key cryptographic algorithm introduced in 1977. Unlike classical ciphers, RSA uses number theory and modular arithmetic to securely encrypt and decrypt messages.

RSA works by:
1. Choosing two prime numbers (p and q)
2. Computing n = p x q
3. Computing φ(n) = (p-1)(q-1)
4. Choosing public key e
5. Computing private key d

Encryption: C = M^e mod n
Decryption: M = C^d mod n
